Projects
1. MyDay - Individual Task Manager
MyDay allows users to create and manage their tasks, track progress and prioritize their to-do list. It is built using Angular in the frontend, Node.js in the backend, and MongoDB as the database.
Features
  1. User Login/Signup: Users can sign up for the application by providing their username, name, email and password. Once they have registered, they can log in to the application using their username and password.
  2. JWT Authorization: The application uses JSON Web Tokens (JWT) for authorization. JWT is a compact, URL-safe means of representing claims to be transferred between two parties. In this application, JWT is used to secure the communication between the frontend and the backend.
  3. Task Viewer: Once logged in, users can create and manage their tasks. They can add a task title and description. They can also track the progress of their tasks and mark them as complete when finished.
Tech Stack: NodeJs, Angular 15, Bootstrap
